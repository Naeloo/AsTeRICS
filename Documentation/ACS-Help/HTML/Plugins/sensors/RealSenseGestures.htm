RealSenseGestures

Component Type: Sensor (Subcategory: Computer Vision)

OS: >= Windows 8.1, 64 bit or Ubuntu 16.04 (or derivative), 64-Bit

Requires AsTeRICS to run in a 64-Bit Java Virtual Machine

This component connects to the Intel Real Sense SR300 and D400 series 3D cameras and provides simple hand gesture recognition functionality. More specifically, only the amount of extended fingers on one hand is tracked. This opens up some interesting possibilities like entering a number or clicking using a fist. To give an idea of how well these use cases work, demos have been included in the bin/ACS/models/useCaseDemos/realSenseGestures folder.

Requirements

You need either an Intel Real Sense SR300 camera or a D400 series camera plugged in and configured.

While Windows should install the drivers for these cameras automatically, there might be other dependencies missing to successfully run the plug-in. For this reason, it is recommended to install the approriate Intel RealSense Depth Manager Tool for your particular camera model.

On Linux, additional configuration of the kernel's webcam driver (uvcvideo) is required. For more information and a guide, refer to the appropriate librealsense wiki page on GitHub.

Note: Intel RealSense cameras can be finnicky as to what computers and software configurations they support. It is recommended to make sure the USB port the camera is plugged into provides plenty of power. Furthermore, the plugin might fail to connect if the camera's firmware version is too low. If errors occur, troubleshooting these issues via Google is your best bet, as many other people have encountered such problems. You can make sure the camera works by opening the Intel RealSense Viewer Tool and enabling the various video streams..

Intel Real Sense F200 (SR300) camera

Output Port Description

  • fingers_extended [integer]: The number of fingers extended on a single hand, ranging from 0 to 5.
  • detected [boolean]: Not yet implemented - If the camera is currently detecting a hand.

Event Trigger Description

This plugin does not trigger any events.

Properties

This plugin does not feature any configurable properties.
